Organizor

Organizor API

Organizor is a multi-tenant ASP.NET Core API for creating, provisioning, resolving, upgrading, and deleting tenant environments.

Documentation Map

Area Use it for

Configuration Connection strings, PostgreSQL settings, logging, CORS, OpenAPI, and Scalar.
Security CORS, admin route exposure, error detail, and destructive operation notes.
Internal Operations Admin routes, tenant onboarding, upgrades, deletion, documentation deployment, and sitemap output.

Solution Structure

  • Code7Dev.Organizor.Api hosts the HTTP API, middleware, OpenAPI/Scalar setup, health checks, CORS, and logging pipeline.
  • Code7Dev.Organizor.Application contains tenant use cases and service contracts.
  • Code7Dev.Organizor.Domain contains the tenant aggregate and lifecycle status.
  • Code7Dev.Organizor.Infrastructure contains EF Core persistence, PostgreSQL database provisioning, tenant database resolution, and migration support.
